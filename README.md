# 🧾 Smart Invoice Generator

An intelligent full-stack web application that automatically generates invoices for products.
The system uses a Machine Learning model to predict product price, GST, and discount, and then calculates the final invoice total instantly.

This project demonstrates the integration of Frontend + Backend + Machine Learning to automate invoice generation.

---

## 🚀 Features

- Enter product name and quantity
- AI predicts product price
- Automatically calculates GST
- Applies discount automatically
- Generates final invoice instantly
- Clean and simple user interface
- Ready for deployment

---

## 🧠 How It Works

1. The user enters a **product name** and **quantity**.
2. The request is sent to the **Flask backend**.
3. A **Machine Learning model** predicts:
   - Product price
   - GST percentage
   - Discount percentage
4. The system calculates:
   - Subtotal
   - GST amount
   - Discount amount
5. The final invoice total is displayed on the webpage.
